# baget-heirloom
Heirloom Marketplace - Next.js 14 App Router

## Tech Stack
- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS** (Configured, using custom Sage/Clay theme)
- **Agent Databases** (PostgreSQL-backed)

## Routes
- `/`: Dynamic marketplace and waitlist
- `/onboard`: Cook registration portal
- `/api/onboard/kitchen`: Secure backend for cook intake
